# 保險不承保情形一覽

## 保險不承保情形一覽

以下彙總 **保本險 / 利增險** 在典型產品設計下**不賠付**，或**走取消 / 退費而非觸發賠付**的情形（與[觸發條件](../../bao-xian/chu-fa-tiao-jian.md)、[退費規則](../../bao-xian/tui-fei-gui-ze.md)一致）。**正式除外責任以用戶簽署的保險條款全文為準**。

***

### 常見不觸發賠付場景

| 场景                    | 说明                                                                                      |
| --------------------- | --------------------------------------------------------------------------------------- |
| **保單到期前未觸及約定條件**      | 進入 `expired`；到期一般**不退**保費（見[退費規則](../../bao-xian/tui-fei-gui-ze.md)）。                   |
| **用戶主動全平**            | 保單隨倉位結束而取消，按 `CLOSE` 等原因退費，**非**「保險事故賠付」。                                               |
| **普通止盈 / 止損先於保險價位觸發** | 平倉原因非保險主導路徑，保單多半取消，並按對應原因退費。                                                            |
| **強平但未走保本險賠付路徑**      | 若未滿足保本險觸發語義，則可能按普通強平處理，保本險**不**按「整筆保證金復活」敘事賠付。                                          |
| **利增險未在保險止盈路徑觸發**     | 若獲利側未按 `INS_TAKE_PROFIT` 等約定歸因，則利增險不進入 `triggered` 賠付鏈。                                 |
| **同倉另一險種已主觸發**        | 單次平倉只有一個主觸發；另一張 active 保單**伴生取消**並退費，而非雙賠（見[觸發條件](../../bao-xian/chu-fa-tiao-jian.md)）。 |

***

### 系統與營運限制

* **新承保熔斷**：通常限制**新單**，是否不撤銷歷史保單仍以正式條款為準。
* **unsupported 市場**：非開放交易對或幣種**不可投保**。
