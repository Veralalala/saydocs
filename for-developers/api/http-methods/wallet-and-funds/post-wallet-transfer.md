# POST wallet/transfer

```
POST /v1/private/exchange/wallet/transfer
```

#### Request Body

| Field         | Type   | Required | Description                                                                                                    |
| ------------- | ------ | -------- | -------------------------------------------------------------------------------------------------------------- |
| `amount`      | string | Yes      | Amount is a positive string of characters passed in ledger precision.                                          |
| `to_address`  | string | Yes      | ToAddress is the destination EVM address and cannot be equal to the current signature address.                 |
| `transfer_id` | string | No       | The TransferID is an optional idempotent key; when omitted, the UUID is automatically generated by the server. |

#### Response

```json
{
  "success": true,
  "data": {
    "tx_id": "..."
  }
}
```

#### Response Fields

| Field        | Type    | Description |
| ------------ | ------- | ----------- |
| `data`       | object  |             |
| `data.tx_id` | string  |             |
| `success`    | boolean |             |
