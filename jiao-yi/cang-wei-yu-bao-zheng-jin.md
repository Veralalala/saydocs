# 倉位與保證金

**倉位** = 你在某個標的、某個方向上的**那一張合約單**：用了多少**保證金**、**均價**多少、**數量**多少、**離強平還有多遠**。每個倉位都**自己管自己的錢**，方便你按**策略**與**風險偏好**分開交易。

**保證金** = 你**押在這筆倉位上**的 **USDC**，用來承受波動、**支付資金費**，也決定這筆倉位**離強平有多遠**。SayFi 採用 **逐倉**。每一口倉位都**自己承擔自己的風險**，不會預設挪用其他倉位的錢來抵抗風險。

***

### 你能在介面上看到什麼

在 **目前持倉**頁中，可用右上角 **All accounts** 篩選**子帳戶 / Trader**。持倉表各欄與產品含義如下：

| 欄名（介面）            | 含義                                                                                                                                                                                           |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **SYMBOL**        | 合約標的，如 **BTC-USD**；旁邊可能帶有**分享 / 節點**類快捷入口，實際以端上為準。                                                                                                                                           |
| **SIDE**          | 倉位方向與槓桿，如 **LONG 10x** / **SHORT 10x**。也就是多 / 空與倍數一同展示。                                                                                                                                      |
| **ACCOUNT**       | 該倉歸屬的 **AI Trader / 子帳戶**。常用**頭像或簡稱**標示。                                                                                                                                                     |
| **POSITION SIZE** | 目前持倉的**數量**。這裡指標的資產數量，不是美元名義。                                                                                                                                                                |
| **MARGIN**        | 本筆倉位**鎖定的保證金**，多以 **USDC** 顯示，如 `$12.87`。                                                                                                                                                    |
| **ENTRY**         | **開倉均價**（ENTRY），作為建倉加權成本參考。                                                                                                                                                                  |
| **MARK**          | **標記價**區域。主行多為**當前 Mark**；載入中或無資料時，可能顯示 **「-」**。同格**副行小字**常標示 **Liq. price**，也就是清算價 / 預估強平價，如 `Liq. price $70157.11`。這表示若 **Mark 朝不利方向**發展到該參考附近，且你沒有補保證金，就可能進入**強平流程**。實際仍以即時計算與 Margin 為準。 |
| **UNREALIZED**    | **未實現盈虧**與**收益率**，如 `0.000 (0.00%)`；會隨 **Mark** 相對 **ENTRY** 的變化而變動。                                                                                                                         |
| **TX HASH**       | 行尾可能含有**鏈上交易**外鏈圖示。                                                                                                                                                                          |
| 操作倉位              | **Partial close**（部分平倉）與 **Close all**（全部平倉）                                                                                                                                                 |

**資金費**會從**這筆倉位的保證金裡**每小時結算一次。在多數配置下，它會依**費率正負與你的方向**決定是**出帳**還是**入帳**，因此你的**有效保證金**會微幅變動。詳見 [資金費](zi-jin-fei.md)。

***

### 為什麼 SayFi 堅持「逐倉」

* **單倉不連坐**：在**逐倉**模式下，一筆倉位虧損**不會**自動動用你**其他倉位**裡鎖著的保證金來補洞。
* **算得清、睡得著**：有子帳戶或多個 Trader 時，**資金與策略**也能物理隔離。

***

### 平倉之後

* **全平**後，該倉位就會**關閉**，盈虧會轉成**已實現**，你可在**對帳單 / 歷史**中追蹤。
* **部分平倉**會按比例**釋放**部分保證金，並結算對應的盈虧與手續費。

***

### **保證金**

1. **初始保證金**\
   開倉時，系統會按**槓桿**把**名義價值**拆成你要鎖進去的保證金。**具體下限**以介面為準。
2. **維持保證金**\
   倉位開著時，你的**保證金 + 浮動盈虧**必須**始終高於**協議要求的**維持線**；低於維持線，就會進入**強平**流程。
3. **強平價**\
   介面上的**預估強平價 / 清算價**，是根據**當前標記價、你的倉位與保證金、維持率檔位**算出的**參考值**。**行情跨檔、資金費、你手動加減倉**都會讓它變動。實際以**最新介面**為準。
