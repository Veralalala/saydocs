# CFD 概覽

SayFi 上的 **USD 本位永續合約**，可理解為不交割現貨、可長期持有的 CFD 式倉位。它是你與協議之間的 **多 / 空對賭標的**。資金與結算以 **USD** 為錨。**核心體驗**可以概括成：用 **自然語言** 與 AI Trader 下指令，用 **逐倉** 管理單筆風險，用 **保險產品** 保護資產、放大收益。詳見 [什麼是 SayFi？](../guan-wu-sayfi/shen-mo-shi-sayfi.md)。

| 能力          | 說明                                                                                                                                    |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **逐倉**      | 每一筆倉位有 **自己的保證金**；**一筆爆倉不會拖垮** 你名下其他倉位的資金。若同帳戶下有多個子帳戶 / 多個 Trader，資金也彼此隔離。詳見 [子帳戶與錢包](../ai-jiao-yi-yuan/zi-zhang-hu-yu-qian-bao.md)。 |
| **做多 / 做空** | 同一標的可同時持有多個**多倉或空倉**，以 USDC 為保證金。                                                                                                     |
| **槓桿**      | 在協議允許範圍內放大名義敞口。常見範圍為 **1x～100x**，實際以各標的與介面為準。                                                                                         |
| **價格透明**    | 系統用 **標記價** 等機制決定盈虧、強平與資金費；**成交價** 在撮合時相對標記價做**小幅滑點**。                                                                                |
| **止盈 / 止損** | 可以在下單時**一併掛上**；成交建倉後，它們會變成**跟著倉位走的保護規則**，觸發後才會平倉。                                                                                     |
