# Sayfi 鏈上瀏覽器

Sayfi 鏈上瀏覽器是 Sayfi Chain 的**官方區塊瀏覽器**。它面向開發者與終端使用者，適合在**不直接呼叫節點 RPC** 的情況下，即時查詢與檢視鏈上交易、地址與區塊。

**瀏覽器位址：** [https://explorer.sayfi.com/](https://explorer.sayfi.com/)

***

### 功能概覽

| 模組   | 說明                 |
| ---- | ------------------ |
| 全域搜尋 | 依地址、交易哈希或區塊高度查詢    |
| 最新交易 | 即時顯示最新鏈上交易，並自動重新整理 |
| 交易詳情 | 查看單筆交易的完整資訊與負載     |
| 地址詳情 | 查看錢包餘額與歷史交易        |

***

### 核心功能

#### 1. 全域搜尋

首頁提供一個統一搜尋框。支援以下查詢類型：

```
依地址 / 交易哈希 / 區塊搜尋
```

| 查詢類型 | 範例格式                                            | 說明        |
| ---- | ----------------------------------------------- | --------- |
| 錢包地址 | `0x50947b54c6240f5f6c6c832468a4fc9ae09e5d09`    | 查看餘額與歷史記錄 |
| 交易哈希 | `0x3ea36306c8b01fb1831c578c65afa57f631f0a19...` | 查看單筆交易    |
| 區塊高度 | `9035608`                                       | 查看區塊詳情    |

***

#### 2. 最新交易

首頁會列出最新的鏈上交易。預設情況下，列表會**每 2 秒自動重新整理一次**。你也可以手動點擊 **Refresh**。

欄位說明：

| 欄位             | 說明                                          |
| -------------- | ------------------------------------------- |
| **HASH**       | 交易哈希，預設截斷顯示，點擊可進入詳情                         |
| **ACTION**     | 操作類型，目前為 `OPEN POSITION` 或 `CLOSE POSITION` |
| **BLOCK**      | 區塊高度                                        |
| **TIME (UTC)** | UTC 時間                                      |
| **ADDRESS**    | 發起交易的錢包地址                                   |
| **AMOUNT**     | 資產數量，**正數**表示流入，**負數**表示流出，單位為 USDC         |

**範例：**

```
HASH                          ACTION          BLOCK     TIME(UTC)   ADDRESS                    AMOUNT
0xae402346...550f7491         OPEN POSITION   9035809   10:59:40    0x50947b54...e09e5d09      -10 USDC
0x3ea36306...ca7e4d56         CLOSE POSITION  9035608   10:59:20    0x50947b54...e09e5d09      9.8205789013 USDC
```

***

#### 3. 交易详情

點擊任意交易哈希，即可開啟交易詳情頁。

**欄位說明：**

| 欄位                   | 說明                                 |
| -------------------- | ---------------------------------- |
| **Transaction Hash** | 完整交易哈希，十六進位格式，支援複製                 |
| **Status**           | 例如 `Completed`、處理中或失敗              |
| **Timestamp**        | 鏈上時間，UTC 格式為 `YYYY-MM-DD HH:mm:ss` |
| **Block**            | 區塊高度，可跳轉到區塊詳情                      |
| **Address**          | 發起地址，可跳轉到地址詳情                      |
| **Action**           | 操作類型，例如 `CLOSE POSITION`           |
| **Amount**           | 數量與資產類型，例如 `9.8205789013 USDC`     |
| **Payload**          | 原始業務資料，JSON 格式                     |

**Payload 範例：**

```json
{
  "asset": "USDC",
  "amount": "9.8205789013"
}
```

**完整交易範例：**

```
Transaction Hash: 0x3ea36306c8b01fb1831c578c65afa57f631f0a19893e14b85abb815bca7e4d56
Status:           Completed
Timestamp:        2026-04-27 10:59:20
Block:            9035608
Address:          0x50947b54c6240f5f6c6c832468a4fc9ae09e5d09
Action:           CLOSE POSITION
Amount:           9.8205789013 USDC
```

***

#### 4. 地址詳情

你可以透過搜尋地址，或點擊地址連結進入地址詳情頁。

**頂部摘要資訊：**

| 欄位                     | 說明            |
| ---------------------- | ------------- |
| **Address**            | 完整地址，支援複製     |
| **Balance**            | 目前餘額，單位為 USDC |
| **Total Transactions** | 歷史交易總數        |

**地址範例：**

```
Address:             0x50947b54c6240f5f6c6c832468a4fc9ae09e5d09
Balance:             6.52466711 USDC
Total Transactions:  17
```

**歷史記錄表欄位：**

| 欄位                 | 說明                                 |
| ------------------ | ---------------------------------- |
| **Tx Hash**        | 交易哈希，支援點擊查看詳情與複製                   |
| **Action**         | `OPEN POSITION` 或 `CLOSE POSITION` |
| **Block**          | 區塊高度，可點擊跳轉                         |
| **DateTime (UTC)** | 時間格式為 `YYYY-MM-DD HH:mm:ss`        |
| **Amount**         | 變動金額，**正數**表示流入，**負數**表示流出         |

***

### 常見使用情境

#### 1. 確認交易是否已上鏈

1. 從錢包或應用中複製交易哈希。
2. 貼到瀏覽器搜尋框中，並執行 **Search**。
3. 檢查 **Status**。如果顯示 `Completed`，表示交易已經在鏈上確認。

#### 2. 查看地址或倉位歷史

1. 搜尋錢包地址。
2. 在地址詳情頁查看 **Total Transactions** 與歷史列表。
3. 結合 `OPEN POSITION` 與 `CLOSE POSITION` 記錄分析資金流與盈虧。

#### 3. 开发与调试

在接入 Sayfi 時，你可以使用瀏覽器確認以下內容：

* 合約互動是否產生預期的鏈上記錄
* **Payload** 結構是否符合預期
* 區塊時間與順序是否正確

***

### 資料說明

#### 金額規則

| 顯示值                 | 含義                     |
| ------------------- | ---------------------- |
| `-10 USDC`          | 使用者支付 10 USDC，通常表示開倉   |
| `9.8205789013 USDC` | 使用者收到該金額，通常表示平倉並包含盈虧結算 |

> 最多可顯示 **10** 位小數。最終數值以鏈上結算結果為準。

#### 時間格式

所有時間均為 **UTC**：

```
列表頁：     HH:mm:ss
詳情頁：     YYYY-MM-DD HH:mm:ss
```

#### 地址格式

標準 EVM 地址格式：

```
0x + 40 個十六進位字元
範例：0x50947b54c6240f5f6c6c832468a4fc9ae09e5d09
```

### 常見問題

**問：交易狀態沒有更新怎麼辦？**

最新交易列表大約會**每 2 秒自動重新整理一次**。如果你正在查看某一筆具體交易，可以手動重新整理頁面，或點擊 **Refresh**。如果交易始終沒有出現，請檢查網路連線，以及該交易是否已成功**廣播**到鏈上。

**問：搜尋地址沒有結果怎麼辦？**

該地址可能**還沒有鏈上活動記錄**。瀏覽器只顯示已經產生鏈上記錄的地址。

**問：Payload 欄位是什麼？**

**Payload** 是附加在交易中的結構化業務資料，格式通常為 JSON，由上層協議寫入。比如 Sayfi 會在其中記錄資產類型、數量與相關欄位。
