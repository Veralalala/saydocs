# 快速開始

> 品牌標語：**Say it, Trade it, Shield it.** — 詳見 [什麼是 SayFi？](guan-wu-sayfi/shen-mo-shi-sayfi.md)。

| 你會在 SayFi 看到什麼 | 一句話                                                                         |
| -------------- | --------------------------------------------------------------------------- |
| 這是什麼產品         | **AI 原生的鏈上永續交易所**：你用**對話**表達意圖，**AI Trader** 協助整理成訂單與倉位操作。                  |
| 誰負責做決策         | **你。** 助手提供分析與參考方案，**實際成交仍須你確認並完成鏈上簽名**；**不是**未授權的自動交易。                     |
| 資產放在哪裡         | **鏈上。** USDC 資產存放在鏈上合約；你也可以建立**多個**獨立 **Trader / 子帳戶**，資金彼此隔離（**最多 10 個**）。 |

### 建議閱讀順序

#### 1. 先了解產品定位

花幾分鐘先看 [**什麼是 SayFi？**](guan-wu-sayfi/shen-mo-shi-sayfi.md)，確認你是否接受這套產品設計。若你特別在意 **保本險 / 利增險** 的保障邊界，也建議先讀清楚官方文件，不要輕信站外說法。

#### 2. 準備環境並登入

* **裝置**：[網頁版](https://trade.sayfi.com/chat) 與**手機瀏覽器**都可使用，實際支援情況以 [官網](https://trade.sayfi.com/chat) 為準。
* **登入方式**
  * **錢包登入**：使用 MetaMask、WalletConnect、Rabby 等 **EVM 錢包**連線並簽名。
* **儲值（選填）**
  * 支援鏈：Ethereum、Base、BSC、Arbitrum
  * 支援幣別：USDC

#### 3. 建立你的第一個 AI Trader

在應用內完成 **招募專屬 AI Trader** 相關流程：設定**名稱 / 頭像**，勾選 **品種、交易節奏、進場風格、風險偏好、停損紀律** 等屬性。部分欄位支援 **+ Custom**。

→ 分步說明見 [**智慧代理設定檔**](ai-jiao-yi-yuan/zhi-hui-dai-li-she-ding-dang.md)。

#### 4. 理解一件事：一個 Trader = 一個子帳戶

每新增一個 AI Trader，通常就對應**一個獨立子帳戶**：**逐倉、保證金與倉位彼此隔離**。切換 Trader，也等於切換一套資金與對話上下文。

→ 詳見 [**子帳戶與錢包**](ai-jiao-yi-yuan/zi-zhang-hu-yu-qian-bao.md)。

#### 5. 用自然語言開始互動

在對話中，你可以**查行情、問規則、下市價 / 限價 / 止盈止損**等。句型可直接照抄，也可以依你的習慣改寫。

→ 見 [**對話指令參考**](ai-jiao-yi-yuan/dui-hua-zhi-ling-can-kao.md)。

* **再提醒一次**：**執行前一定要在介面上確認**；不要把**私鑰或助記詞**交給任何人，包括自稱客服的人。

#### 6. 要深入時，再看主文件

以下主題**不適合**在快速開始裡一次講完，但只要你打算實際交易，建議至少先快速看過：

* [**訂單類型、倉位管理、價格與資金費**](jiao-yi/)
* [**保證金、強制平倉、風控機制**](jiao-yi/)
* [**交易保險**](bao-xian/)**：保本險 / 利增險**
* [**常见问题**](https://app.gitbook.com/s/2ydpQ0rXK8hfDMCn6VFp/readme/chang-jian-wen-ti)

### 每個人都該先知道的安全提醒

* **SayFi 不提供傳統人工客服**；凡是透過電話或私訊索取**轉帳、私鑰、助記詞、付費解凍**的，一律視為詐騙。
* 協助資訊以**官方文件、應用內說明與社群公開資訊**為主。
* 若對外宣傳提到 **0 門檻 / 0 手續費 / 保本金 / 保獲利** 等說法，**細節與適用邊界一律以文件為準**。
