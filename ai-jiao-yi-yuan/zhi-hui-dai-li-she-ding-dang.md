# 智慧代理設定檔

在 SayFi 裡，每一位 **AI Trader** 不只是一次對話，而是一份可長期使用的 **智慧交易代理設定檔**：有名稱、頭像、可勾選的**屬性標籤**，以及獨立的對話記憶邊界。你可以把它理解成**專屬於你的交易助手**——先在設定檔裡定好「關注哪些市場、偏好哪種節奏與進場邏輯、風險與停損紀律」，再在對話裡用自然語言下指令。

***

### 在應用裡：招募專屬 AI Trader

建立流程在介面中通常會以**招募第&#x20;**_**N**_**&#x20;位專屬 AI Trader** 之類的方式呈現，引導你依序完成：

1. **上方：名稱與頭像** — 為代理命名，並選擇或編輯顯示頭像，建立一眼就能辨識的身份。
2. **下方：屬性標籤** — 依分類勾選或自訂，組合出這位 Trader 的「交易風格與偏好」。
3. **開始使用** — 儲存設定後，進入與該 Trader 綁定的對話與**子帳戶**。

提示文案大意是：**先在上方設好名稱與頭像，再在下方選好屬性標籤，就可以開始。** 實際文案與步驟**以目前版本的應用介面為準**。

#### 名稱與頭像

| 項目               | 說明                                         |
| ---------------- | ------------------------------------------ |
| **AI Trader 名稱** | 自訂名稱，方便在多個智慧代理之間切換；介面通常會有**字數上限**，請不要超過限制。 |
| **頭像**           | 可選擇或編輯的顯示圖片，只影響**辨識與視覺呈現**，不會改變鏈上權限。       |

***

### 屬性標籤說明

以下分類與選項是依產品 **UI 能力**整理而來。**標籤文案、是否可多選、是否提供「+ Custom」**，請以你裝置上的介面為準。這些設定會影響 AI 在**分析、提醒與方案建議**時的著重方向；**實際下單與調倉仍須你的明確指令或確認**。

#### 偏好品種（通常可多選）

助手會**優先關注**你勾選的市場，方便結合行情與資訊做討論：

* **Cryptocurrency**（加密貨幣）
* **Commodities**（大宗商品，例如金屬等，實際支援以平台交易對為準）
* **Energy**（能源）
* **Stocks**（股票類標的，以平台支援的 USDC 永續為準）
* **Forex**（外匯等，以平台是否上架為準）

多選代表這些市場類別都會進入該 Trader 的**關注與討論範圍**，不代表系統會自動跨市場同時下單；是否開倉，仍由你決定。

#### 交易節奏

| 選項       | 文件中的理解方式（不代表報酬承諾）                    |
| -------- | ------------------------------------ |
| **Low**  | 偏向**低頻 / 長線視角**，較少進出，傾向等待較大級別的結構或事件。 |
| **Mid**  | **中等**頻率，平衡訊號品質與交易次數。                |
| **High** | 更關注**短線節奏與機會密度**，對話中可能更頻繁討論進出場時機。    |

**注意**：這裡描述的是**互動與分析節奏的偏好**，**不是**未授權的高頻自動下單；鏈上成交仍須你確認並簽名。

#### 進場風格（通常可多選，也可能支援 + Custom）

常見選項例如：

* **Trend Follow** — 顺势思路
* **Range** — 區間震盪內的來回操作思路
* **Breakout** — 突破後跟進
* **Pullback Buy** — 拉回承接類型的進場討論

部分介面的 **Entry Style** 會提供 **「+ Custom」**：你可以補充自己的進場邏輯關鍵字或短語，讓助手更貼近你的習慣。**自訂內容仍須落在合規與產品能力範圍內，請以介面可用選項為準。**

#### 風險偏好（可能支援 + Custom）

| 選項               | 說明                                                     |
| ---------------- | ------------------------------------------------------ |
| **Conservative** | 保守，強調資本保全與較小的風險曝險。                                     |
| **Balanced**     | 平衡，在收益與風險之間取中。                                         |
| **Aggressive**   | 積極，對話中可能更常討論**高波動**情境下的倉位與槓桿想法，但**實際槓桿上限仍以協議與標的風控為準**。 |

若提供 **+ Custom**，可用來細化你對風險的文字描述，並與下方的停損紀律搭配使用。

#### 停損紀律（可能支援 + Custom）

| 選項           | 文件中的理解方式                         |
| ------------ | -------------------------------- |
| **Strict**   | 更強調**硬停損、紀律出場**，建議中偏向明確價位或機械式規則。 |
| **Flexible** | 更允許**移動停損、分批停利 / 回撤出場**等彈性討論。    |

若支援 **+ Custom**，你可以補充自己對「多嚴格才算嚴格」的判準；執行層面仍以你在介面設定的 **TP/SL** 與鏈上結果為準。

### 這是客製化交易助手，不是一般聊天 Agent

* **行為一致**：在同一份設定檔下，助手會結合你已勾選的**品種、節奏、進場、風險、停損**偏好，減少「每次都要重講一次原則」的摩擦。
* **多策略分倉**：你可以擁有**多個**獨立 Trader，並與**子帳戶**一一對應。像是一個偏加密短線，一個偏商品長線；每份設定檔都維護**獨立對話與上下文**，策略不容易混在一起。
* **你仍然掌舵**：不管標籤怎麼設定，**實際下單與調倉都需要你的明確指令或確認**；助手提供的是理解與執行協助，**不是**放著不管的自動交易。

***

### 和子帳戶的關係

每個 **Agent（AI Trader）** 都對應**一個**鏈上子帳戶：資金、逐倉倉位與保證金池彼此隔離。切換 Trader / 智慧代理，等於切換到**另一套資金與風險邊界**上的對話。詳情見 [子帳戶與錢包](zi-zhang-hu-yu-qian-bao.md)。

***

### 使用建議

* 新使用者可以先**招募 1 個** Trader，先把開倉與平倉流程走過一次，再依不同策略建立第 2、第 3 個，最多 **10 個**子帳戶。
* 多選標籤時，**少而精**通常比全部勾選更容易讓助手聚焦。若你使用 **+ Custom**，盡量寫成**可執行、可核對的**短句。
* 如果你大幅調整風險或停損類標籤，建議先用**小部位**驗證對話建議是否符合預期，再逐步放大倉位。
